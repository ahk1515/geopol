🔐 Connexion GitHub
Un token personnel (PAT) est requis pour piloter le pipeline ETL, modifier la configuration et déposer des fichiers semi-automatiques. Il est stocké uniquement dans ce navigateur.
Comment créer le token (une seule fois) :
  1. Aller sur github.com → Settings → Developer settings
  2. Personal access tokens → Fine-grained tokens → Generate new token
  3. Repository access : seulement ahk1515/geopol
  4. Permissions : Contents: Read & Write + Actions: Read & Write
  5. Coller le token ci-dessous (commence par github_pat_)
